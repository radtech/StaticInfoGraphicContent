AI Innovation Lab Framework

Age-appropriate, hands-on experiments connecting AI to standard science subjects.

🔍

Class 6: Vision & Recognition

Introduction to computer vision through physical-to-digital mapping.

  • ➤ The Botanist AI: Use webcam-based classification to identify monocot vs dicot leaves (Biology Link).
  • ➤ Shadow Classifier: Map light angles and intensity to machine learning detection thresholds (Physics Link).
🖥

Class 7: Pattern & Prediction

Transitioning from observations to algorithmic "If-Then" logic using block-based AI.

  • ➤ Weather Predictor: Build a logic block that predicts humidity based on evaporation patterns (Physics Link).
  • ➤ Mode Master: Identify frequency trends in dataset samples to predict "most likely" outcomes (Math Link).
💻

Class 8: Modeling & Weights

Introduction to parameters, model training, and basic Python script interaction.

  • ➤ Linear Growth Model: Predict crop yield by adjusting "weights" for water vs sun in a Python script (Bio/Math Link).
  • ➤ Robotic Logic: Use sensor-input datasets to model simple force-response actuators (Physics Link).
